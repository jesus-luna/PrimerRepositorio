Mi primer repositorio

1) Crear carpeta donde trabajar

2) Escribir #git init para iniciar repositorio

3) Crear archivos en la carpeta

4) Escribir en terminal #git status

5) #git add . para agregar archivos al repositorio

6) #git commit -m "identificador" para saber como se van llamando neustros cambios

6) Creo un nuevo repositorio en github

6) #git remote add origin https://github.com/jesus-luna/PrimerRepositorio.git esto depende el repositorio en el que este

6) #git push -u origin master esto es para subir los commits al repositorio
